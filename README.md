# We Run

React + Vite 기반의 팀 프로젝트 시작용 빈 템플릿입니다.

## 실행 방법

PowerShell에서 `npm` 실행이 막히면 `npm.cmd`를 사용하세요.

```bash
npm.cmd install
npm.cmd run dev
```

브라우저에서 터미널에 표시되는 로컬 주소로 접속하면 됩니다.

## 현재 구성

- `index.html`: React 앱 진입점
- `src/main.jsx`: 빈 앱 컴포넌트
- `src/styles.css`: 공통 초기 스타일
